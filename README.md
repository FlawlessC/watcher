# my_list_app aka watcher 

This is my first project, although still a fun one.

## Getting Started

The essence of this application is to clear out all your bookmarks with a bunch of plans to watch/play/read, etc.
The app is convenient for tracking progress if you abandon something. You can log in via Google account (in the future I will add update via Google Play). Well, you can conveniently filter different types of leisure (and add your own). There is also a dark and light theme, a magic random button (it's set to follow your filters at the moment)
